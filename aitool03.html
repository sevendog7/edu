# 지침
- 선정 기준은 해당 시간 범위 내에 새롭게 발생했거나 의미 있는 진전이 확인된 사안으로 한다.
- 공공성, 사회적 파급력, 보도량을 종합해 중요한 순서대로 고른다.
- 같은 주제의 유사한 보도는 하나의 항목으로 묶고, 대표 사례 중심으로 정리한다.
- 해당 시간 범위 내에 적합한 이슈가 3건 미만이면 확인된 건수만 보고한다.
- 부족한 항목 수를 억지로 채우지 않는다.
- 제공된 맥락이나 확인된 정보에만 근거해 작성하며, 확인되지 않은 내용은 추정하여 쓰지 않는다.
- 필요한 정보가 부족해 판단이 어려운 경우에는 추측하지 말고, 확인 가능한 범위만 명시한다.
- 간결하고 정보 밀도 높게 작성하며, 같은 내용을 반복하지 않는다.

4. 문제점
5. 해결방안
6. 용어 설명(새로운 용어가 있는 경우에만 추가)
# 맥락
